# Cursor Ops · MØDLR Tools

Cursor Ops is a precision 3D Cursor workflow system for Blender — fast, modal, deterministic, and production-ready.  
Part of the MØDLR Tools suite.

---

## Purpose of This Repository

This repository is used for:

- Issue tracking  
- Bug reports  
- Feature requests  
- Release notes  

It is not the sales or distribution platform.

---

## Installation

1. Download the official ZIP release.
2. Open Blender.
3. Go to **Edit → Preferences → Add-ons → Install**.
4. Select the ZIP file.
5. Enable **Cursor Ops**.

## Compatibility

Minimum supported version: Blender 3.3 LTS.  
Future versions are expected to work but are not guaranteed until tested.

Specifically tested on:

- 3.3.19 LTS  
- 3.6.21 LTS  
- 4.2.18 LTS  
- 4.5.7 LTS  
- 5.0.1
