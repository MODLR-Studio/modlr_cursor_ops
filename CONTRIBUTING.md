# Contributing

Thanks for your interest in Cursor Ops.

## Scope of This Repository

This repository is used for:

- Issue tracking  
- Bug reports  
- Feature requests  
- Public release notes  

It is not the primary development repository.

## Code Contributions

At this time, this repository does **not** accept external code contributions or pull requests.

If you would like to suggest improvements or report issues, please open an Issue instead.

## Reporting Issues

When submitting a bug report, please include:

- Blender version  
- Cursor Ops version  
- Operating system  
- Clear reproduction steps  
- Screenshots or screen recordings if possible  

Well-documented reports help resolve issues faster.

## Code of Conduct

By participating, you agree to abide by the Code of Conduct in this repository.
